## About

DemonQuest is a 3rd Person RPG built with Unity 3D.  It features a fully animated arena with combat mechanics, advanced shading, and even some easter eggs.

## 🎬 Start Screen

* Logo inspired by Possessed
* Buttons inspired by Runescape
* Created with Adobe After Effects
